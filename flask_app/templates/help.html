Lab Monitor Help

Arm/Disarm:

"Arm" means the alarm is active and can send SMS notifications. "Disarm" means it is currently turned off.

Hysteresis:

When the temperature exceeds the trigger threshold, the alarm will continue to send messages until the temperature drops below the trigger threshold minus the hysteresis value.

Repeat Alerts:

If turned off, the system sends an alert SMS only once. If turned on, it will keep sending SMS alerts at the set interval.

Send Daily Report:

Sends an SMS report at the set time every day to users who have opted in for daily reports.

Power Source:

Indicates whether the system is running on 120-AC power or UPS power if there is no main power supply.

Add Phone Number:

Send daily status report: Check this box if you want this number to receive daily reports from the alarm system.

Grant Administrator Privileges: Check this box if you want this number to have the ability to configure the alarm settings via messages.
